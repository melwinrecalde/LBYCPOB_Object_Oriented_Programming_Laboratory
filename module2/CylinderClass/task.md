**Cylinder Class**

Implement the following UML Class:
```text
+----------------------------------+
|            Cylinder              |
+----------------------------------+
| - radius: double                 |
| - height: double                 |
+----------------------------------+
| + Cylinder(double, double)       |
| + getRadius(): double            |
| + setRadius(double): void        |
| + getHeight(): double            |
| + setHeight(double): void        |
| + calculateVolume(): double      |
| + calculateSurfaceArea(): double |
+----------------------------------+
```

**Note**

- (-) means private; while (+) means public
- The constructor accepts radius and height as its arguments
