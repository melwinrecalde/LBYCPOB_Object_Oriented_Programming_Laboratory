**OOP Grade and Ranking Tracker**

Develop and build a menu-driven console application that serves as a simplified academic grade tracker with OOP design.
This program allows a teacher or administrator to input a class roster, record subject scores for each
student, compute individual averages, determine numeric grades, generate class statistics (highest average,
lowest average, class mean), and print a formatted summary report. Some classes need no modification since they are 
working as intended. The classes to be modified are as follows:
- GradeCalculator
- IDVerifier
- Student
- StudentRepository

The rest of the classes need no modification or a simple uncomment to some useful code blocks. The student is required
to handle these minor modification on their own.

Refer to the LBYCPOB Laboratory Manual for more details.
