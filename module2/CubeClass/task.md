**Cube Class**

Implement the following UML Class:
```text
+-----------------------------+
|            Cube             |
+-----------------------------+
| - side: double              |
+-----------------------------+
| + Cube(double)              |
| + getSide(): double         |
| + setSide(double): void     |
| + calculateSurfaceArea():   |
|   double                    |
| + calculateVolume(): double |
+-----------------------------+
```

**Note**

- (-) means private; while (+) means public
- The constructor accepts side as its argument
