**Circle Class**

Implement the following UML Class:
```text
+------------------------+
|         Circle         |
+------------------------+
| - radius: double       |
+------------------------+
| + Circle(double)       |
| + getRadius(): double  |
| + setRadius(double):   |
|   void                 |
| + calculateArea():     |
|   double               |
| + calculatePerimeter():|
|   double               |
| + calculateDiameter(): |
|   double               |
+------------------------+
```

**Note**

- (-) means private; while (+) means public
- The constructor accepts radius as its argument
