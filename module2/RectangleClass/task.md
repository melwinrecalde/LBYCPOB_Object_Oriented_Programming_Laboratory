**Rectangle Class**

Implement the following UML Class:
```text
+--------------------------+
|        Rectangle         |
+--------------------------+
| - width: double          |
| - height: double         |
+--------------------------+
| + Rectangle(double,      |
|   double)                |
| + getWidth(): double     |
| + setWidth(double): void |
| + getHeight(): double    |
| + setHeight(double): void|
| + calculateArea(): double|
| + calculatePerimeter():  |
|   double                 |
+--------------------------+
```

**Note**

- (-) means private; while (+) means public
- The constructor accepts width and height as its arguments
