**Square Class**

Implement the following UML Class:
```text
+------------------------+
|         Square         |
+------------------------+
| - side: double         |
+------------------------+
| + Square(double)       |
| + getSide(): double    |
| + setSide(double): void|
| + calculateArea():     |
|   double               |
| + calculatePerimeter():|
|   double               |
+------------------------+
```

**Note**

- (-) means private; while (+) means public
- The constructor accepts side as its argument
