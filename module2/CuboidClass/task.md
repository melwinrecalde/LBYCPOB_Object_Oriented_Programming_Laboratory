**Cuboid Class**

Implement the following UML Class:
```text
+----------------------------------+
|             Cuboid               |
+----------------------------------+
| - length: double                 |
| - width: double                  |
| - height: double                 |
+----------------------------------+
| + Cuboid(double, double, double) |
| + getLength(): double            |
| + setLength(double): void        |
| + getWidth(): double             |
| + setWidth(double): void         |
| + getHeight(): double            |
| + setHeight(double): void        |
| + calculateVolume(): double      |
| + calculateSurfaceArea(): double |
+----------------------------------+
```

**Note**

- (-) means private; while (+) means public
- The constructor accepts the length, width, and height as its argument
