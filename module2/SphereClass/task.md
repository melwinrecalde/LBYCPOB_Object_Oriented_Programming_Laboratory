**Sphere Class**

Implement the following UML Class:
```text
+-------------------------------+
|            Sphere             |
+-------------------------------+
| - radius: double              |
+-------------------------------+
| + Sphere(double)              |
| + getRadius(): double         |
| + setRadius(double): void     |
| + calculateVolume(): double   |
| + calculateSurfaceArea():     |
|   double                      |
+-------------------------------+
```

**Note**

- (-) means private; while (+) means public
- The constructor accepts radius as its argument
