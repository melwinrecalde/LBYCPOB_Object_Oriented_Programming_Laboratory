**Data Types Practice**

Declare and initialize the following five Java data types with appropriate values:
- `int` - store your age (e.g., 20)
- `double` - store a decimal number (e.g., 3.14)
- `char` - store a single letter (e.g., 'A')
- `boolean` - store a true/false value (e.g., true)
- `String` - store your name (e.g., "John")

After declaring each variable, print its value to the console. Each value should be printed on a new line.

**Example Output**
```text
20
3.14
A
true
Juan
```
**Requirements**
- Declare one variable of each type: `int`, `double`, `char`, `boolean`, `String`
- Assign any valid value to each variable
- Print each variable's value on a separate line
- Use `IO.println()` or `System.out.println()` for each print statement
