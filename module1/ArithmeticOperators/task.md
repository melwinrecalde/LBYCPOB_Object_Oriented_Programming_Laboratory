**Arithmetic Operators Practice**

Write a Java program that demonstrates the use of common arithmetic operators and increment/decrement operators.

Perform the following operations using two integer variables (e.g., `a = 10` and `b = 3`):

1. Addition (`+`) - add the two numbers
2. Subtraction (`-`) - subtract the second from the first
3. Multiplication (`*`) - multiply the two numbers
4. Division (`/`) - divide the first by the second
5. Modulo (`%`) - find the remainder of the first divided by the second
6. Increment (`++`) - increase the first variable by 1
7. Decrement (`--`) - decrease the second variable by 1

Print the result of each operation on a separate line.

**Example Output**
```text
13
7
30
3
1
11
2
```
**Explanation of Output:**
- Line 1: 10 + 3 = 13
- Line 2: 10 - 3 = 7
- Line 3: 10 * 3 = 30
- Line 4: 10 / 3 = 3
- Line 5: 10 % 3 = 1
- Line 6: 10++ becomes 11
- Line 7: 3-- becomes 2

**Requirements:**
1. Declare two integer variables: `a = 10` and `b = 3`
2. Perform all five arithmetic operations (+, -, *, /, %)
3. Perform increment on the first variable
4. Perform decrement on the second variable
5. Print each result on a new line
6. Total of 7 lines of output only
