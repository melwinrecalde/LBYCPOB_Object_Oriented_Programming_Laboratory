**Comparison and Logical Operators Practice**

Write a Java program that demonstrates the use of comparison operators and logical operators.

Create two integer variables (e.g., `x = 10` and `y = 5`). Then evaluate and print the results of the following expressions:

**Comparison Operators:**
1. Equal to (`x == y`)
2. Not equal to (`x != y`)
3. Greater than (`x > y`)
4. Less than (`x < y`)
5. Greater than or equal to (`x >= y`)
6. Less than or equal to (`x <= y`)

**Logical Operators:**
7. Logical AND (`(x > y) && (y > 0)`)
8. Logical OR (`(x > y) || (y > x)`)
9. Logical NOT (`!(x == y)`)

Print each result on a separate line. (Note: Follow the sequence strictly.)

**Example Output**
```text
false
true
true
false
true
false
true
true
true
```

**Explanation of Output:**
- Line 1: 10 == 5 is false
- Line 2: 10 != 5 is true
- Line 3: 10 > 5 is true
- Line 4: 10 < 5 is false
- Line 5: 10 >= 5 is true
- Line 6: 10 <= 5 is false
- Line 7: (10 > 5) AND (5 > 0) is true
- Line 8: (10 > 5) OR (5 > 10) is true
- Line 9: NOT(10 == 5) is true

**Requirements:**
1. Declare two integer variables: `x = 10` and `y = 5`
2. Use all six comparison operators (==, !=, >, <, >=, <=)
3. Use all three logical operators (&&, ||, !)
4. Print each result on a new line 
5. Total of 9 lines of output
