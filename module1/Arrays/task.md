**Arrays Practice**

Write a Java program that demonstrates how to declare, initialize, and traverse arrays.

Your program should perform the following tasks:

**Part 1: Integer Array**
- Declare and initialize an integer array with values: 10, 20, 30, 40, 50
- Use a for-loop to traverse and print each value

**Part 2: String Array**
- Declare and initialize a String array with names: "Alice", "Bob", "Charlie"
- Use a for-loop to traverse and print each name

**Part 3: Sum of Array Elements**
- Declare and initialize an integer array: 5, 10, 15, 20
- Calculate and print the sum of all elements

**Part 4: Find Maximum Value**
- Declare and initialize an integer array: 8, 3, 12, 5, 9
- Find and print the maximum value in the array

**Expected Output**
```text
Integer Array 
10
20
30
40
50

String Array 
Alice
Bob
Charlie

Sum of Array 
Sum: 50

Maximum Value 
Maximum: 12
```

**Requirements:**
1. Declare and initialize an integer array with 5 values
2. Declare and initialize a String array with 3 names
3. Use loops to traverse both arrays and print values
4. Calculate sum of integer array elements
5. Find maximum value in an integer array 
6. Print results with section headers as shown

**Hints:**
- Array declaration: `int[] numbers = {10, 20, 30, 40, 50};`
- Get array length: `array.length`
- For-loop traversal: `for (int i = 0; i < array.length; i++)`
- Enhanced for-loop: `for (int num : numbers)`
- Sum: add each element to a sum variable
- Maximum: compare each element with current max
